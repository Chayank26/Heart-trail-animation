# Heart Trail Animation 

A simple and interactive heart trail animation built using HTML, CSS, and JavaScript.
Hearts follow the mouse cursor creating a smooth trailing effect.

## Features
- Mouse-based heart trail animation
- Smooth fade-out effect
- Lightweight and responsive
- No external libraries used

## Tech Stack
- HTML
- CSS
- JavaScript

## What I Learned
- DOM manipulation
- Event handling (mousemove)
- Dynamic element creation and removal
- Basic animations using CSS and JavaScript

## How to Run
Open `index.html` in a browser or use Live Server.
